# Module 5: AI编程与AI产品核心概念

---

**所属模块**: Module 5  
**学习难度**: ★★☆☆☆ ~ ★★★★☆ (入门到中上)  
**建议学时**: 45小时  
**前置知识**: Module 1-4 基础内容  
**后续学习**: Module 6 - AI产品实战

---

## 📚 模块概述

本模块系统介绍AI编程与AI产品的核心概念，帮助学习者从产品经理视角理解AI技术的原理、应用和局限。内容涵盖AI/ML/DL的关系、大语言模型工作原理、代码生成工具、提示词工程、RAG架构、Agent智能体、Embedding与向量数据库、Function Calling、流式输出、Token计费等关键技术概念，以及AI产品评估、幻觉问题、安全合规和持续优化等产品管理实践。

---

## 🎯 学习目标

完成本模块后，你将能够：

1. 用通俗语言向非技术团队解释AI核心概念
2. 评估不同AI技术方案的产品适用性
3. 设计合理的AI产品架构和功能流程
4. 建立AI产品评估、安全和持续优化的方法论
5. 与开发团队高效沟通AI技术需求和方案

1. **按顺序学习**：本模块内容有递进关系，建议按目录顺序学习
