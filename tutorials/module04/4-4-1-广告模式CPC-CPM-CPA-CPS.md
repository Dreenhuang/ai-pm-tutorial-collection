## 核心内容

### 一、广告模式的商业逻辑

**通俗比喻：** 广告模式就像"出租广告牌"。你有一个好位置（用户的注意力），广告主想在这里展示信息。关键问题是：按什么标准收费？

**广告变现的核心三角：**

```
         用户
       /      \
     注意力    数据
     /          \
   平台 ◄────► 广告主
   (流量)       (预算)
```

| 角色 | 核心诉求 | 衡量标准 |
|------|----------|----------|
| **平台** | 最大化广告收入 | RPM（千次展示收入）、eCPM |
| **广告主** | 最小化获客成本 | CPA、ROI |
| **用户** | 最小化广告干扰 | 广告频率、相关性 |

### 二、四种主要计费模式

#### 1. CPM（Cost Per Mille，千次展示费用）

**定义：** 广告每展示 1000 次，广告主支付的费用。

**通俗理解：** 不管有没有人看，只要广告出现在屏幕上就收费。就像高速公路上的广告牌——不管有多少人经过，租金是固定的。

| 维度 | 说明 |
|------|------|
| **适合谁** | 品牌广告主（追求曝光） |
| **计费单位** | 元 / 1000 次展示 |
| **典型价格** | 开屏广告 50-200 元/千次，信息流 10-50 元/千次 |
| **优点** | 收入可预测，平台风险低 |
| **缺点** | 不保证效果，广告主风险高 |

**计算公式：**
```
CPM = 广告费用 / 展示次数 × 1000
```
示例：1000 元广告费，获得了 50000 次展示 → CPM = 1000 / 50 = 20 元

#### 2. CPC（Cost Per Click，单次点击费用）

**定义：** 用户每点击一次广告，广告主支付的费用。

**通俗理解：** 只有当用户"感兴趣"并点击了才收费。就像发传单——只有接过传单的人，你才需要给发传单的人付费。

| 维度 | 说明 |
|------|------|
| **适合谁** | 效果广告主（追求引流） |
| **计费单位** | 元 / 次点击 |
| **典型价格** | 搜索广告 1-10 元/次，信息流 0.5-3 元/次 |
| **优点** | 广告主风险低，按效果付费 |
| **缺点** | 平台承担点击率风险 |

**关键指标：CTR（点击率）**
```
CTR = 点击次数 / 展示次数 × 100%
```

**CTR 健康标准：**

| 广告类型 | CTR 健康范围 |
|----------|-------------|
| 搜索广告 | 3-8% |
| 信息流广告 | 1-3% |
| 开屏广告 | 0.5-2% |
| Banner 广告 | 0.1-0.5% |

#### 3. CPA（Cost Per Action，单次行为费用）

**定义：** 用户完成特定行为（注册、下载、购买）后，广告主支付的费用。

**通俗理解：** 只有当用户"真的买了"才收费。就像房产中介——只有房子成交了才收佣金。

| 维度 | 说明 |
|------|------|
| **适合谁** | 效果导向的广告主（追求转化） |
| **计费单位** | 元 / 次行为 |
| **典型行为** | 注册、下载、表单提交、试用 |
| **优点** | 广告主风险最低，完全按效果付费 |
| **缺点** | 平台风险最高，转化受产品本身影响 |

#### 4. CPS（Cost Per Sale，单次销售费用）

**定义：** 用户通过广告完成实际购买后，按销售额的一定比例支付佣金。

**通俗理解：** 卖出多少，分多少。就像"代销"——你帮我卖东西，卖出去了才分钱。

| 维度 | 说明 |
|------|------|
| **适合谁** | 电商广告主、联盟营销 |
| **计费单位** | 销售额的百分比 |
| **典型比例** | 1-10%（品类不同差异大） |
| **优点** | 广告主零风险，按实际收入分成 |
| **缺点** | 平台需要追踪完整购买链路 |

### 三、四种模式的对比

| 对比维度 | CPM | CPC | CPA | CPS |
|----------|-----|-----|-----|-----|
| **广告主风险** | 最高 | 中 | 低 | 最低 |
| **平台风险** | 最低 | 中 | 高 | 最高 |
| **平台收益上限** | 固定 | 取决于 CTR | 取决于转化率 | 取决于客单价 |
| **适合阶段** | 品牌曝光 | 引流获客 | 效果转化 | 电商变现 |
| **代表平台** | 传统媒体、开屏广告 | Google Ads、百度竞价 | 应用下载广告 | 淘宝客、京东联盟 |

**模式演进逻辑：**
```
CPM（按展示收费） → CPC（按点击收费） → CPA（按行为收费） → CPS（按销售收费）
广告主风险越来越高，平台风险越来越低
```

### 四、广告产品的设计要素

#### 广告位类型

| 广告位 | 位置 | 特点 | eCPM 范围 |
|--------|------|------|-----------|
| **开屏广告** | App 启动时 | 曝光量大，干扰性强 | 50-200 元 |
| **信息流广告** | 内容流中 | 原生性强，点击率高 | 10-50 元 |
| **Banner 广告** | 页面顶部/底部 | 曝光量大，点击率低 | 5-20 元 |
| **激励视频** | 游戏中主动选择观看 | 用户主动，完播率高 | 30-100 元 |
| **搜索广告** | 搜索结果中 | 意图明确，转化率最高 | 50-500 元 |
| **原生广告** | 融入内容形式 | 不易察觉，体验好 | 15-40 元 |

#### 广告产品关键指标

| 指标 | 定义 | 计算公式 | 健康标准 |
|------|------|----------|----------|
| **eCPM** | 有效千次展示收入 | 总收入 / 展示次数 × 1000 | 因产品类型而异 |
| **填充率** | 有广告展示的次数 / 请求次数 | 填充次数 / 请求次数 | >80% |
| **CTR** | 点击率 | 点击 / 展示 | >2% 为佳 |
| **CVR** | 转化率 | 转化 / 点击 | >5% 为佳 |
| **ARPU** | 每用户广告收入 | 广告总收入 / DAU | 持续增长 |
| **广告渗透率** | 看到广告的用户比例 | 有广告曝光用户 / DAU | 40-70% |

### 五、广告模式与用户体验的平衡

**核心矛盾：** 广告越多，收入越高，但用户体验越差，用户可能流失。

**平衡策略：**

| 策略 | 做法 | 示例 |
|------|------|------|
| **控制频次** | 限制用户每天看到广告的次数 | 抖音每刷 10 条内容出 1 条广告 |
| **提升相关性** | 推荐用户感兴趣的广告 | 根据浏览历史推荐相关商品 |
| **原生化** | 广告长得像内容 | 小红书"种草笔记" |
| **用户选择** | 允许用户关闭或减少广告 | YouTube Premium 免广告 |
| **价值交换** | 看广告获得好处 | 激励视频得游戏道具 |

**产品经理红线：**
- 不要用"暗模式"（Dark Pattern）诱导用户点击广告
- 广告不能影响产品的核心功能
- 广告内容必须经过审核，不能展示违规内容
- 给用户提供"为什么看到这个广告"的解释

## 案例分析

### 案例一：微信的广告策略

**微信的广告位：**

| 广告位 | 形式 | 计费模式 | 特点 |
|--------|------|----------|------|
| 朋友圈广告 | 原生信息流 | CPM | 社交属性，信任度高 |
| 公众号广告 | 文中/底部 | CPC/CPM | 内容场景，精准匹配 |
| 小程序广告 | Banner/激励视频 | CPM/CPC | 功能场景，转化率高 |
| 视频号广告 | 信息流/直播 | CPM | 视频场景，沉浸感强 |

**策略分析：**
- 微信严格控制广告数量（朋友圈广告每天不超过 2 条）
- 利用社交关系链提升广告相关性（"朋友也在看"）
- 公众号广告与内容高度融合，降低打扰感

**数据参考：** 微信 2023 年广告收入约 1000 亿元，主要靠朋友圈和公众号广告驱动。

### 案例二：抖音的广告变现

**抖音的广告生态：**

| 广告类型 | 适用场景 | 计费模式 |
|----------|----------|----------|
| 开屏广告 | 品牌曝光 | CPM |
| 信息流广告 | 效果转化 | CPC/CPM |
| 搜索广告 | 精准获客 | CPC |
| 直播广告 | 电商转化 | CPS |
| 挑战赛 | 品牌互动 | CPM + 内容合作 |

**抖音的优势：**
- 推荐算法天然适合广告投放（内容 + 广告一起推荐）
- 用户停留时长长，广告曝光量大
- 短视频形式广告创意空间大

**产品经理启示：** 抖音的广告策略是"广告即内容"。好的广告本身就是一条有趣的短视频，用户甚至愿意主动看完。这是广告体验的最高境界。

## 产品经理视角的关键思考

1. **广告模式的核心是"注意力货币化"。** 产品经理需要思考：用户的注意力值多少钱？广告主的预算从哪来？平台能分多少？这三个问题的答案决定了广告模式的可持续性。

2. **不同广告位有不同的"价格天花板"。** 搜索广告最贵（用户意图明确），信息流中等（用户被动接受），Banner 最便宜（用户习惯性忽略）。选择广告位就是选择价格天花板。

3. **广告质量比广告数量重要。** 100 条垃圾广告的总收入可能不如 1 条精准广告。提升广告的相关性和创意质量，是提升 eCPM 的最有效方法。

4. **广告模式的终局是"看不见的广告"。** 最好的广告是用户不觉得它是广告。原生广告、内容营销、品牌植入——这些形式的广告体验最好，但对平台的内容生产能力要求最高。

## 思考问题

1. 一个工具类 App（如计算器、天气）想通过广告变现，你会选择哪些广告位？为什么不能用信息流广告？

2. 某产品的 eCPM 从 30 元下降到 15 元，可能的原因有哪些？作为产品经理，你会如何排查和优化？

3. 比较微信和抖音的广告策略差异。为什么微信朋友圈广告控制很严，而抖音的信息流广告密度更高？
